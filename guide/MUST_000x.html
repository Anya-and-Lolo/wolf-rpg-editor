◆ Uninstalling WOLF RPG Editor

[Learn How To]

　Uninstall WOLF RPG Editor.


[1]　Deleting the program

　Simply delete the entire "en-US Woditor"        
　folder. WOLF RPG Editor does not use the
　Windows Registry, so removing the folder
　completely uninstalls the program and
　restores your system to the state it was
　in before installation.

　Thank you very much for downloading and
　using WOLF RPG Editor.
