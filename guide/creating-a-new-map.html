Essential Knowledge

Creating a new map

Create a new map called "Test" (with the filename test.mps), then draw the letters "RPG" using grass tiles as explained below, and save the map.

Author: SmokingWOLF

※ Note: Completing this step alone will not allow the player to reach this map. You will also need to complete the next tutorial: Moving the player to another map.
1

Create a new map

First, click on this icon in the top bar to create a new map.

2

Check map settings

The "Create New Map" window will appear as shown on the right.

3

Name your map

For now, just fill in the fields highlighted by the red boxes.

Set the filename to "test (.mps)" and the map name to "Test."

When done, click "OK".

※ Note: Make sure the filename does not duplicate an existing map file. If you accidentally use the same filename as another map, the original may be overwritten and lost. If "test" is already taken, try "test2", "test3," or something similar.
4

Great work!

Your new map has been created!

As you can see, the entire map is black, so let’s draw something on it.

5

Draw a simple map

Click Layer 1 to open the Map Tile window, as shown on the right.

6

Pick a map tile

Select a grass tile from the Map Tile window. Then click anywhere in the black area on the left to place the tile.

Now try drawing a map as if you were a painter creating a picture!

7

Paint your map

Place lots of grass tiles and write the letters "RPG!"

When you're done, click Save icon to save your new map.

Your first map is complete. Great work!

Tip

I placed a tile in the wrong spot!

If you tried to write "RPG!" but placed a tile incorrectly, don't worry.

The top-left tile acts as an eraser. Select the black square in the upper- left corner of the Map Tile window, then click the area you want to erase.

Alternatively, press X while drawing.

Extra Note

Even after creating the map, it may not immediately appear in the Map Selection window . If that happens, click "Refresh" to reload the map list.

The new map "4: Test" will appear as shown on the right.
