◆ Creating your first event

[Goal]

　In the Sample Game's "Sample Map A" we'll create a new chicken event
　that says "Cock-a-doodle-doo♪" when the player talks to it.

　※ What is an event?
　　For now, you can think of an event as "an NPC in an RPG villager that does or says
　　something when you interact with it". That's all you need to know for now.

[1]　Switch the map

　Open the Map Selection window by clicking the    
　 icon and select "1: Sample Map A".
　
[2]　Switch the map layer

　Switch to the Event Layer .
　Right-click near the area just above the tranfer    
　destination position. A menu like the one shown
　on the right will appear.

　Select "Create/Edit Event".
[3]　Create new event

　The Event Settings window will appear, as shown 
　on the right.

　
[4]　Set the event sprite

　Let's set the event's walking sprite first. To do        
　that, double-click the area highlighted in
　red.
[5]　Change the facing direction

　A window like the one on the right will appear.        
　Select "Animal_Chicken.png" and click "OK".


　※ You can click the character image in the
　　right-hand preview area to change the
　　direction the character is facing.
[6]　Done!

　The chicken sprite has now been successfully          
　assigned to an event.
[7]　Make it speak

　Next, let's add some dialogue for the chicken.         

　Click the button in the lower-right corner of
　the Event Window:
　"■ Command Input Window ■"
[8]　

　The "Insert Event Command" window will open.
　Make sure the selected tab is "Show Message",
　as shown on the right.
[9]　

　Type "Cock-a-doodle-doo♪" in the text input field,
　then click the "Input" button.
[10]

　If everything looks like the image on the right,
　you've succeeded.

　The Trigger Condition is already set to
　"Execute on Confirm key",
　so this event will work as a
　talk-triggered event right away.
[11]　

　Click the "Save (Entire Map)" button
　at the bottom-right of the Event window,
　then click "Test Play"
　to save this map (with the new event)
　and start a test play.
[12]　

　Select "New Game" on the title screen,
　skip through the intro, then
　press the Confirm key while facing the chicken
　to talk to it!

　It said "Cock-a-doodle-doo♪"!

　Your first event is complete!
　Great work!




　　

[Side Note]
　Can I put anything I want in the event "Name" field?

　When you set a graphic, the filename is automatically entered
　in the "Name" field, but you can change it
　to anything — "Chicken", "Cock-a-doodle-doo Event", whatever you like.

　So what is the "Name" actually used for?
　While editing the Event Layer, hovering the cursor over an event
　will show that event's "Name" in the title bar.
　
　That's all it does — so a cool name won't get you anything!
